# To start project

1. Clone Repo
2. `npm i`
3. `node server.js` to run the mock server
4. `npm run start` to start react server

### Tech stack

- `apollo-server-express` for mock server
- `apollo-client`
- `create-react-app`
- `tachyons`

### What's missing

- Only used Typescript in a couple of files
- No jest testing
- No sort
